## SvelteKit Blog Starter

A opinioned Markdown blog starter built with SvelteKit, MDSveX,
Tailwind CSS and DaisyUI.

Put your Markdown docs in the `/posts` directory.

Give them the frontmatter:

```markdown
---
date: 2021-09-17
title: Hello world!
published: true
---
```
